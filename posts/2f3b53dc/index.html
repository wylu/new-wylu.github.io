Hexo简介

Hexo 是一个快速、简洁且高效的博客框架，基于 Node.js 实现。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内即可利用靓丽的主题生成静态网页。

安装Git

  • Linux (Fedora, Red Hat, CentOS)： sudo yum install git-core
